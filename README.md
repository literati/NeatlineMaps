

NeatlineMaps was created by the [Scholars' Lab][1]  at the University of Virginia Libtary. It connects the powerful open-source geospatial server [ http://www.geoserver.org][2] and Omeka. It permits users to ingest georeferenced images into Omeka and use them as parts of interactive maps.

  Installation 

----------

*  Download and configure GeoServer (see [GeoServer Documentation][3]).

*  Upload and activate the NeatlineMaps plugin (see [Installing_a_Plugin][4])

  Usage 

----------

*  Add a georeferenced tiff as a "Map" data type
